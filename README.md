# 🤖 STM32智能小车项目

这是一个基于 **STM32F103C8T6**微控制器的智能小车项目，具备遥控核心功能。

## 🚀 核心功能

* **[功能1]：** [蓝牙遥控]，通过手机 APP 或手柄控制小车运动。
* **[功能2]：** [蓝牙遥控]，通过手机 APP 或手柄控制两个舵机运动。
* **[功能3]：** [蓝牙遥控]，通过手机 APP 或手柄控制水枪发射。

---

### 开发环境

* **单片机型号：** `STM32F103C8T6`
* **开发工具：** STM32CubeMX+Keil MDK-ARM V5
* **HAL 库版本：** STM32F1c8t6 hal
* **编译语言：** C

###蓝牙控制

*上舵机：按下键3000800（3代表调动上面电机开始运动，080代表方向，可自己调试）松开键0000000
*下舵机：按下键2000800（2代表调动上面电机开始运动，080代表方向，可自己调试）松开键0000000
*前进：按下键0010960 松开键0000000
*后退：按下键0000900 松开键0000000
*左转：按下键0010201 松开键0000000
*右转：按下键0010202 松开键0000000

云台/机械臂控制：
20004560：控制第二组舵机到45°位置
30007890：控制第三组舵机到78°位置
小车运动控制：
0011000：小车以速度110前进（直行）
0011001：小车以速度110前进并左转
0011002：小车以速度110前进并右转
0111000：小车以速度110前进（仅右轮工作，原地旋转？）
小车后退控制：
0101000：小车以速度101后退（直行后退）
0101001：小车以速度101后退并左转（但代码中后退时转向可能不生效）
停止与特殊功能：
0000000：完全停止所有电机
4444444：开启发射/抓取装置（DRIVE1置高）
5555555：关闭发射/抓取装置（DRIVE1置低）
