# 🤖 STM32智能小车项目

这是一个基于 **STM32F103C8T6**微控制器的智能小车项目，具备遥控核心功能。

## 🚀 核心功能

* **[功能1]：** [蓝牙遥控]，通过手机 APP 或手柄控制小车运动。
* **[功能2]：** [蓝牙遥控]，通过手机 APP 或手柄控制两个舵机运动。
* **[功能3]：** [蓝牙遥控]，通过手机 APP 或手柄控制水枪发射。

---

### 开发环境

* **单片机型号：** `STM32F103C8T6`
* **开发工具：** STM32CubeMX+Keil MDK-ARM V5
* **HAL 库版本：** STM32F1c8t6 hal
* **编译语言：** C
