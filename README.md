# 🤖 STM32智能小车项目

这是一个基于 **STM32F103C8T6** 微控制器的智能小车项目，具备蓝牙遥控、云台控制和发射装置等功能。

##  演示视频
https://www.douyin.com/user/self?modal_id=7583356570161417482
---

## 🚀 核心功能

* **蓝牙遥控控制**：通过手机APP或手柄控制小车运动
* **云台/机械臂控制**：通过蓝牙控制上下两个舵机的运动
* **水枪/发射装置控制**：通过蓝牙控制水枪发射功能

## 📱 蓝牙控制协议

### 🎮 云台控制指令

| 指令 | 功能描述 | 按下指令 | 松开指令 |
|------|----------|----------|----------|
| **上舵机控制** | 控制上方舵机运动 | `3000800` | `0000000` |
| **下舵机控制** | 控制下方舵机运动 | `2000800` | `0000000` |

### 🚗 基础运动控制

| 动作 | 按下指令 | 松开指令 |
|------|----------|----------|
| **前进** | `0010960` | `0000000` |
| **后退** | `0000900` | `0000000` |
| **左转** | `0010201` | `0000000` |
| **右转** | `0010202` | `0000000` |

### 🔧 高级控制指令

#### 云台/机械臂精确控制
```
20004560：控制第二组舵机到45°位置
30007890：控制第三组舵机到78°位置
```

#### 小车运动控制
```
0011000：小车以速度110前进（直行）
0011001：小车以速度110前进并左转
0011002：小车以速度110前进并右转
0111000：小车以速度110前进（特殊模式，仅右轮工作）
```

#### 小车后退控制
```
0101000：小车以速度101后退（直行后退）
0101001：小车以速度101后退并左转
```

#### 停止与特殊功能
```
0000000：完全停止所有电机
4444444：开启发射/抓取装置（DRIVE1置高）
5555555：关闭发射/抓取装置（DRIVE1置低）
```

## ⚙️ 使用说明

1. **硬件准备**：按照接线图连接各模块
2. **固件烧录**：使用Keil MDK编译并下载程序到STM32
3. **蓝牙连接**：通过手机APP连接蓝牙模块
4. **遥控操作**：使用上述指令控制小车各项功能

## 📝 注意事项

* 舵机角度值可根据实际需求调整
* 后退时的转向功能可能需要根据具体硬件调整
* 确保电源供应稳定，避免电压波动影响控制精度
